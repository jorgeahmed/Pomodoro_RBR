  2. Auth → Providers: activa Google y/o GitHub
  3. Auth → URL Config: agrega https://jorgeahmed.github.io/Pomodoro_RBR
  4. SQL Editor: crea tabla subscribers (ver plan)
  5. Settings → API: copia URL y anon key abajo
